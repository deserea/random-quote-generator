# Random Quote Generator

A simple Python tool that fetches random quotes using an API, processes JSON data, and saves results to a file.

## What it does
- Sends requests to a public API
- Receives data in JSON format
- Extracts and prints quotes and authors
- Accepts user input for multiple quotes
- Saves quotes to a text file
- Handles request errors gracefully

## Technologies used
- Python
- Requests library
- JSON
- API integration
